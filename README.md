# TextFormatter


TextFormatter is an Text Editor built with Angular Framework , where you can simply Input your text and get your text reversed , capitalized, Remove White Spaced , Remove Special Characters features .

With above function this TextFormatter also provides you the live Word Count and Character count on your inputs.

In addition to ,it also provide styling to your fonts like Bold, Italic and Underlined with option of increasing and decreasing the font size .
